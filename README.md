# TagIO

Advanced TagLib wrapper for Node.JS

## Status

**This module is under hard development!**

| ---------- | ---------------- |
| Generic    | done             |

| FLAC       | XIPH,ID3v1,ID3v2 |

## Prerequisites

* Installed C++ compiler (GCC, CLANG)
* Installed cmake (version 2.8 or above)
