# TagIO

Advanced TagLib wrapper for Node.JS

**This module is under hard development!**

### Implemented

* **Generic File Tags** - complete
* **MPEG File** - generic API, ID3v2


## Prerequisites

* Mac or Linux OS
* Installed cmake
* Installed taglib
